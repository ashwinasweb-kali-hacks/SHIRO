Netlify Deployment Guide

To use Shiro AI on Netlify, you must allow CORS in Ollama:

  1. Set Env Var: OLLAMA_ORIGINS
  2. Value: https://your-site.netlify.app (or *)
  3. Restart Ollama completely.

For remote access, use Ngrok or Cloudflare Tunnel.
